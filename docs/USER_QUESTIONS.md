## 2. Vercel 배포 관련

### 2.1 배포 절차 및 MCP 연동
**질문**: vercel을 이용하여 배포하려해
1. 절차를 알려줘
2. vercel을 MCP를 이용하여 이곳 cursor ai에 반영할 수 있나, 그렇게하면 조금 더 쉽게 배포할 수 있을까

### 2.2 여러 프로젝트 배포
**질문**: 가령, vercel을 이용하여 하나의 프로젝트를 배포했어. 그리고 나서 새로운 프로젝트를 vercel로 배포할 수 있나

### 2.3 빌드 명령어
**질문**: npm run build 하기 전에 cd web 해야하나

### 2.4 배포 버전 확인
**질문**: 지금 배포되고 있는것이 dev 버전인가 production버전인가

### 2.5 Vercel 설정 관련
**질문**: 
- vercel프로젝트중의 화면인데, public repository name으로 해야할지 아니면 private repository name 으로 해야할지
- vercel에서 smart_farm_web 프로젝트를 대시보드의 설정을 통해 삭제했는데, "Add New" 버튼을 클릭하면 여전히 남아있어
- vercel 설정 진행중 위와 같은 메시지 (Invalid request: should NOT have additional property 'rootDirectory')
